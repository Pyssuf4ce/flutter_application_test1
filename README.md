## ✨ ฟีเจอร์เด่น
*   💬 **ระบบแชทเรียลไทม์:** ติดต่อผู้ซื้อ-ผู้ขายได้ทันใจ พร้อมระบบแจ้งเตือน
*   📦 **จัดการสินค้าอัจฉริยะ:** ลงขายง่าย จัดการสต็อกและรูปภาพได้ครบวงจร
*   🔐 **ความปลอดภัยสูง:** มั่นใจด้วยระบบจัดการผู้ใช้ผ่าน Supabase Auth
*   🎨 **ดีไซน์ระดับพรีเมียม:** อินเทอร์เฟซที่ลื่นไหล รองรับทั้งโหมดสว่างและมืด

## 🛠️ เทคโนโลยีที่ใช้
*   **Frontend:** Flutter & Dart
*   **Backend:** Supabase (Auth, Database, Storage)
*   **Local DB:** Hive
*   **Typography:** Google Fonts (Manrope, Inter)

---

## 🚀 เริ่มต้นใช้งาน
1. `git clone https://github.com/Pyssuf4ce/flutter_application_test1.git`
2. `flutter pub get`
3. ตั้งค่า Supabase ในไฟล์ `lib/main.dart`
4. `flutter run`
