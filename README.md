# VAULT Marketplace 🛍️
VAULT เป็นแอปพลิเคชัน Marketplace ยุคใหม่ที่ออกแบบมาเพื่อความรวดเร็ว สวยงาม และใช้งานง่าย โดยใช้ Flutter และ Supabase เป็นระบบหลังบ้าน

## 🚀 ฟีเจอร์เด่น
- **Real-time Chat:** ระบบแชทพูดคุยระหว่างผู้ซื้อและผู้ขายแบบเรียลไทม์
- **Product Listing:** ลงประกาศขายสินค้าได้ง่าย พร้อมระบบจัดการรูปภาพ
- **Secure Auth:** ระบบสมาชิกที่ปลอดภัยผ่าน Supabase Auth
- **Modern UI:** การออกแบบที่เน้น UX/UI ที่ทันสมัยและรองรับการใช้งานที่ลื่นไหล

## 🛠️ Tech Stack
- **Frontend:** Flutter (Dart)
- **Backend/Database:** Supabase
- **Local Storage:** Hive
- **State Management:** Flutter Hooks / Stateful Widgets
- **Design:** Google Fonts (Manrope, Prompt)

---

## 🏁 เริ่มต้นใช้งาน
1. Clone repository นี้ลงเครื่อง
2. รันคำสั่ง `flutter pub get` เพื่อติดตั้ง dependencies
3. ตั้งค่า Supabase Key ใน `lib/main.dart`
4. สั่ง `flutter run` เพื่อเริ่มใช้งานแอป
