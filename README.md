## ✨ Features
*   💬 **Real-time Communication:** Chat instantly with buyers and sellers.
*   📦 **Smart Listing:** Upload and manage your inventory with ease.
*   🔐 **Military-Grade Auth:** Secure login and profile management via Supabase.
*   🎨 **Premium UI/UX:** A modern, dark-themed aesthetic that feels high-end.

## 🛠️ Tech Stack
*   **Mobile App:** Flutter & Dart
*   **Backend Cloud:** Supabase (Auth, DB, Storage)
*   **Caching:** Hive (Local Storage)
*   **Design System:** Google Fonts (Manrope, Inter)

---

## 🚀 Getting Started
1. `git clone https://github.com/your-username/vault-marketplace.git`
2. `flutter pub get`
3. Configure your Supabase credentials in `lib/main.dart`
4. `flutter run`
